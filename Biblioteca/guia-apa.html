Opciones de Integración

Enlace Directo

La forma más sencilla de integrar esta guía es mediante un enlace directo desde tu menú principal o sección de recursos:

<a href="https://tudominio.com/guia-apa" aria-label="Guía APA 7">
  <i class="fas fa-book"></i> Guía de Citado APA 7
</a>

También puedes usar un botón destacado:

<button onclick="location.href='https://tudominio.com/guia-apa'" class="btn">
  <i class="fas fa-graduation-cap"></i> Ver Guía APA 7
</button>

Iframe

Para incrustar la guía dentro de otra página de tu sitio:

<iframe src="https://tudominio.com/guia-apa"
  title="Guía APA 7 - Universidad UARM"
  width="100%"
  height="600px"
  frameborder="0"
  allow="fullscreen">
</iframe>

Recomendado para:

  • Plataformas de aprendizaje (LMS)
  • Portales de recursos académicos
  • Sitios con estructura de paneles

API JSON

Para desarrolladores que necesitan integrar contenido específico:

// Obtener datos de ejemplo de citación
fetch('https://tudominio.com/api/apa-examples')
  .then(response => response.json())
  .then(data => {
    console.log(data);
  });

Endpoints disponibles:

  • /api/apa-examples - Ejemplos de citación
  • /api/apa-templates - Plantillas de referencias
  • /api/apa-faqs - Preguntas frecuentes
